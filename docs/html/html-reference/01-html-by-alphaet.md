# مرجع عناصر HTML

بیا یکم باهم بررسی کنیم که تگ‌های HTML چی هستن و چه کارایی‌هایی دارن:

| تگ | توضیحات |
|-----|------------|
| `<!--...-->` | یک کامنت رو تعریف می‌کنه |
| `<!DOCTYPE>` | نوع سند رو تعیین می‌کنه |
| `<a>` | یک لینک رو تعریف می‌کنه |
| `<abbr>` | یک کوتاه‌نویس یا کرنوم رو تعریف می‌کنه |
| `<acronym>` | در HTML5 پشتیبانی نمی‌شه. به جایش از `<abbr>` استفاده کن. |
| `<address>` | اطلاعات تماس برای نویسنده/صاحب سند رو تعریف می‌کنه |
| `<applet>` | در HTML5 پشتیبانی نمی‌شه. به جایش از `<embed>` یا `<object>` استفاده کن. |
| `<area>` | یک ناحیه درون نقشه‌ی تصویری رو تعریف می‌کنه |
| `<article>` | یک مقاله رو تعریف می‌کنه |
| `<aside>` | محتوا رو به جزوه‌های محتوای صفحه می‌ده |
| `<audio>` | محتوای صوتی رو تعریف می‌کنه |
| `<b>` | متن رو توی صفحه ضخیم می‌کنه |
| `<base>` | آدرس پایه/هدف برای تمامی URLهای نسبتی در یک سند رو مشخص می‌کنه |
| `<basefont>` | در HTML5 پشتیبانی نمی‌شه. از CSS به جایش استفاده کن. |
| `<bdi>` | بخشی از متن رو که ممکنه در جهت متفاوتی نسبت به متن دیگه درونش فرمت داده شه رو مجزا می‌کنه |
| `<bdo>` | جهت متن فعلی رو نادیده می‌گیره |
| `<big>` | در HTML5 پشتیبانی نمی‌شه. از CSS به جایش استفاده کن. |
| `<blockquote>` | بخشی رو که از یک منبع دیگه نقل شده رو تعریف می‌کنه |
| `<body>` | بدنه‌ی سند رو تعیین می‌کنه |
| `<br>` | یک خط جدید رو تعیین می‌کنه |
| `<button>` | یک دکمه‌ی قابل کلیک رو تعیین می‌کنه |
| `<canvas>` | برای ترسیم گرافیک‌ها، به صورت داینامیک، از طریق اسکریپتینگ (معمولاً جاوااسکریپت) استفاده میشه |
| `<caption>` | عنوان یک جدول رو تعیین می‌کنه |
| `<center>` | در HTML5 پشتیبانی نمی‌شه. از CSS به جایش استفاده کن. |
| `<cite>` | عنوان یک اثر رو تعیین می‌کنه |
| `<code>` | یک قسمت از کد کامپیوتری رو تعیین می‌کنه |
| `<col>` | ویژگی‌های ستون رو برای هر ستون درون یک عنصر `<colgroup>` مشخص می‌کنه |
| `<colgroup>` | یک گروه از یک یا چند ستون در یک جدول برای قالب بندی مشخص می‌کنه |
| `<data>` | یک ترجمه قابل خوانش ماشینی از یک محتوای خاص رو اضافه می‌کنه |
| `<datalist>` | یک لیست از گزینه‌های پیش‌تعریف شده برای کنترلهای ورودی رو مشخص می‌کنه |
| `<dd>` | یک توضیح/مقدار برای یک مترادف در یک فهرست توضیحی رو تعریف می‌کنه |
| `<del>` | متنی رو که از یک سند حذف شده رو تعریف می‌کنه |
| `<details>` | جزئیات اضافی رو که کاربر می‌تونه ببینه یا مخفی کنه رو تعریف می‌کنه |
| `<dfn>` | یک واژه‌ی خاص که در داخل محتوا تعریف میشه رو مشخص می‌کنه |
| `<dialog>` | یک باکس یا پنجره گفت‌و‌گو رو تعریف می‌کنه |
| `<dir>` | در HTML5 پشتیبانی نمی‌شه. به جایش از `<ul>` به جایش استفاده کن. |
| `<div>` | یک بخش در یک سند رو تعریف می‌کنه |
| `<dl>` | یک لیست توضیحی رو تعریف می‌کنه |
| `<dt>` | یک تعریف/نام در یک فهرست توضیحی رو تعریف می‌کنه |
| `<em>` | متن رو تاکید می‌کنه |
| `<embed>` | یک کانتینر برای یک برنامه خارجی رو تعریف می‌کنه |
| `<fieldset>` | عناصر مرتبط رو در یک فرم گروه‌بندی می‌کنه |
| `<figcaption>` | یک عنوان برای یک عنصر `<figure>` رو تعریف می‌کنه |
| `<figure>` | محتوای مستقل رو مشخص می‌کنه |
| `<font>` | در HTML5 پشتیبانی نمی‌شه. از CSS به جایش استفاده کن. |
| `<footer>` | یک پا رو برای یک سند یا بخش تعریف می‌کنه |
| `<form>` | یک فرم HTML برای ورود کاربر تعریف می‌کنه |
| `<frame>` | در HTML5 پشتیبانی نمی‌شه. |
| `<frameset>` | در HTML5 پشتیبانی نمی‌شه. |
| `<h1> تا <h6>` | عناوین HTML رو تعریف می‌کنه |
| `<head>` | حاوی اطلاعات/فراداده‌های سند هست |
| `<header>` | یک سربرگ برای یک سند یا بخش تعریف می‌کنه |
| `<hgroup>` | یک سربرگ و محتوای مرتبط رو تعریف می‌کنه |
| `<hr>` | یک تغییر موضوعی در محتوا رو تعریف می‌کنه |
| `<html>` | ریشه‌ی یک سند HTML رو تعیین می‌کنه |
| `<i>` | یک قسمت از متن رو با صدای یا حالت مختلف تعریف می‌کنه |
| `<iframe>` | یک فریم درونی رو تعریف می‌کنه |
| `<img>` | یک تصویر رو تعریف می‌کنه |
| `<input>` | یک کنترل ورودی رو تعریف می‌کنه |
| `<ins>` | متنی که به یک سند اضافه شده رو تعریف می‌کنه |
| `<kbd>` | ورودی کیبورد رو تعریف می‌کنه |
| `<label>` | برای یک عنصر `<input>` یک برچسب رو تعریف می‌کنه |
| `<legend>` | یک عنوان برای یک عنصر `<fieldset>` رو تعریف می‌کنه |
| `<li>` | یک مورد لیست رو تعریف می‌کنه |
| `<link>` | رابطه بین یک سند و یک منبع خارجی رو تعریف می‌کنه (اکثراً برای لینک به صفحه‌های استایل استفاده میشه) |
| `<main>` | محتوای اصلی یک سند رو تعیین می‌کنه |
| `<map>` | یک نقشه تصویری رو تعریف می‌کنه |
| `<mark>` | متن رو مشخص/برجسته می‌کنه |
| `<menu>` | یک لیست نامرتب رو تعریف می‌کنه |
| `<meta>` | متاداده‌هایی رو درباره‌ی یک سند HTML تعیین می‌کنه |
| `<meter>` | یک اندازه‌گیری مقیاسی در یک محدوده مشخص (یک گیج) رو تعریف می‌کنه |
| `<nav>` | لینک‌های مسیریابی رو تعریف می‌کنه |
| `<noframes>` | در HTML5 پشتیبانی نمی‌شه. |
| `<noscript>` | یک محتوای جایگزین برای کاربرانی که اسکریپت‌های سمت کلاینت رو پشتیبانی نمی‌کنند رو تعریف می‌کنه |
| `<object>` | یک کانتینر برای یک برنامه خارجی رو تعریف می‌کنه |
| `<ol>` | یک لیست مرتب رو تعریف می‌کنه |
| `<optgroup>` | یک گروه از گزینه‌های مرتبط در یک لیست کشویی رو تعریف می‌کنه |
| `<option>` | یک گزینه در یک لیست کشویی رو تعریف می‌کنه |
| `<output>` | نتیجه‌ی یک محاسبه رو تعریف می‌کنه |
| `<p>` | یک پاراگراف رو تعریف می‌کنه |
| `<param>` | یک پارامتر برای یک شی رو تعریف می‌کنه |
| `<picture>` | یک کانتینر برای چندین منبع تصویری رو تعریف می‌کنه |
| `<pre>` | متن پیش‌فرمت شده رو تعریف می‌کنه |
| `<progress>` | پیشرفت یک کار رو نشون می‌ده |
| `<q>` | یک نقل قول کوتاه رو تعریف می‌کنه |
| `<rp>` | چیزی که باید در مرورگرهایی که حاشیه‌های روبی رو پشتیبانی نمی‌کنند نشون داده بشه رو تعریف می‌کنه |
| `<rt>` | یک توضیح/تلفظی از کاراکترها (برای تایپوگرافی شرقی) رو تعریف می‌کنه |
| `<ruby>` | یک حاشیه‌ی روبی (برای تایپوگرافی شرقی) رو تعریف می‌کنه |
| `<s>` | متنی که دیگر صحیح نیست رو تعریف می‌کنه |
| `<samp>` | خروجی نمونه از یک برنامه کامپیوتری رو تعریف می‌کنه |
| `<script>` | یک اسکریپت سمت کلاینت رو تعریف می‌کنه |
| `<search>` | یک بخش جستجو رو تعریف می‌کنه |
| `<section>` | یک بخش در یک سند رو تعریف می‌کنه |
| `<select>` | یک لیست کشویی رو تعریف می‌کنه |
| `<small>` | متن رو کوچکتر می‌کنه |
| `<source>` | چندین منبع رسانه‌ای رو برای عناصر رسانه‌ای (`<video>` و `<audio>`) تعریف می‌کنه‌|
| `<span>` | یک بخش در یک سند رو تعریف می‌کنه |
| `<strike>` | در HTML5 پشتیبانی نمی‌شه. از `<del>` یا `<s>` به جایش استفاده کن. متن را با خط خورده تعریف می‌کنه |
| `<strong>` | متن مهم را تعریف می‌کنه |
| `<style>` | اطلاعات استایل برای یک سند را تعریف می‌کنه |
| `<sub>` | متن زیرنویس را تعریف می‌کنه |
| `<summary>` | یک عنوان قابل مشاهده برای یک عنصر `<details>` را تعریف می‌کنه |
| `<sup>` | متن بالانویس را تعریف می‌کنه |
| `<svg>` | یک کانتینر برای گرافیک‌های SVG را تعریف می‌کنه |
| `<table>` | یک جدول را تعریف می‌کنه |
| `<tbody>` | محتوای بدنه را در یک جدول گروه‌بندی می‌کنه |
| `<td>` | یک سلول در یک جدول را تعریف می‌کنه |
| `<template>` | یک کانتینر برای محتوایی که باید هنگام بارگیری صفحه مخفی شود را تعریف می‌کنه |
| `<textarea>` | یک کنترل ورودی چندخطی (محدوده متن) را تعریف می‌کنه |
| `<tfoot>` | محتوای پا را در یک جدول گروه‌بندی می‌کنه |
| `<th>` | یک سلول هدر در یک جدول را تعریف می‌کنه |
| `<thead>` | محتوای هدر را در یک جدول گروه‌بندی می‌کنه |
| `<time>` | زمان یا تاریخ مشخص را تعریف می‌کنه |
| `<title>` | عنوانی برای سند را تعریف می‌کنه |
| `<tr>` | یک ردیف در یک جدول را تعریف می‌کنه |
| `<track>` | مسیرهای متنی برای عناصر رسانه‌ای را  (`<video>` و `<audio>`)تعریف می‌کنه |
| `<tt>` | در HTML5 پشتیبانی نمی‌شود. به جای آن از CSS استفاده کنید. متن تله‌نویس را تعریف می‌کنه |
| `<u>` | متنی را که بیان نشده و از متن عادی متفاوت است، تعریف می‌کنه |
| `<ul>` | یک لیست نامرتب را تعریف می‌کنه |
| `<var>` | یک متغیر را تعریف می‌کنه |
| `<video>` | محتوای ویدیویی را تعریف می‌کنه |
| `<wbr>` | یک شکست خط ممکن را تعریف می‌کنه |
