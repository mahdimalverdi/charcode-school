# مقدمه‌ای درباره HTML

HTML یه زبان کده که همه استفاده می‌کنن تا صفحه‌های وب بسازن. با استفاده از HTML، می‌تونی ساختار و محتوای صفحات وب رو تعریف کنی و اون‌ها رو برای مرورگرها قابل فهم کنی.


## HTML چیه؟
* HTML مخفف Hyper Text Markup Language هست.
* یه زبان استاندارد برای ساخت صفحات وب.
* ساختار یه صفحه وب رو توصیف می‌کنه و از سری عناصر تشکیل شده.
* این عناصر به مرورگر می‌گن چطوری محتوا رو نمایش بده.
* مثلاً "این یه عنوانه"، "این یه پاراگرافه"، "این یه لینکه"، و غیره.

## یک سند ساده HTML
```html
<!DOCTYPE html>
<html>
<head>
<title>عنوان صفحه</title>
</head>
<body>

<h1>عنوان اول من</h1>
<p>پاراگراف اول من .</p>

</body>
</html>
```



### توضیحات مثال

- اولی `<!DOCTYPE html>` می‌گه که این سند HTML5 هست.
- عنصر `<html>` پایه‌ایه صفحه HTML.
- عنصر `<head>` اطلاعات جزئی درباره صفحه HTML رو داره.
- عنصر `<title>` عنوان صفحه HTML رو مشخص می‌کنه (که توی نوار عنوان مرورگر یا تب صفحه نشون داده میشه).
- عنصر `<body>` بدنه‌ی سند رو تعریف می‌کنه، واسه تمام محتواهای قابل دیدن مثل عناصر عنوان، پاراگراف‌ها، تصاویر، پیوندها، جداول، لیست‌ها، و غیره.
- عنصر `<h1>` یه عنوان بزرگ رو تعریف می‌کنه.
- عنصر `<p>` یه پاراگراف رو مشخص می‌کنه.

## چیه عنصر HTML؟
یه عنصر HTML با یک برچسب شروع، محتوا و یک برچسب پایانی تعریف می‌شه:
```html
<برچسب>محتوا</برچسب>
```

محتوای عنصر HTML همه چیز از برچسب شروع تا برچسب پایانی است:

```html
<h1> عنوان اول من </h1>

<p> پاراگراف اول من </p>
```

:::tip[توجه]
توجه: یه بخشی از عناصر HTML محتوا ندارن مثل عنصر `<br>`. این عناصر به عنوان عناصر خالی شناخته می‌شوند و بدون برچسب پایانی هستن!
:::

## وب‌مرورگرها
هدف یه وب‌مرورگر (مثل Chrome، Edge، Firefox، Safari) اینه که اسناد HTML رو بخونه و درست نشون بده.

مرورگر HTML برچسب‌ها رو نمایش نمی‌ده، ولی ازشون برای فهمیدن چطوری سند رو نمایش بده استفاده می‌کنه:

# ساختار صفحه HTML
در زیر یه تصویر از ساختار یه صفحه HTML داریم:

<div style={{width:'99%',border:'1px solid grey',padding:'3px',margin:0, direction:'ltr', background:'#E7E9EB'}}>
        &lt;html&gt;
    <div style={{width:'90%',border:'1px solid grey',padding:'3px',margin:'20px'}}>
        &lt;head&gt;
        <div style={{width:'90%',border:'1px solid grey',padding:'5px',margin:'20px'}}>
        &lt;title&gt;Page title&lt;/title&gt;
        </div>
        &lt;/head&gt;
    </div>
    <div style={{width:'90%',border:'1px solid grey',padding:'3px',margin:'20px'}}>
        &lt;body&gt;
        <div style={{width:'90%',border:'1px solid grey',padding:'3px',margin:'20px',background:'#fff'}}>
            <div style={{width:'90%',border:'1px solid grey',padding:'5px',margin:'20px'}}>&lt;h1&gt;This is a heading&lt;/h1&gt;</div>
            <div style={{width:'90%',border:'1px solid grey',padding:'5px',margin:'20px'}}>&lt;p&gt;This is a paragraph.&lt;/p&gt;</div>
            <div style={{width:'90%',border:'1px solid grey',padding:'5px',margin:'20px'}}>&lt;p&gt;This is another paragraph.&lt;/p&gt;</div>
        </div>
        &lt;/body&gt;
    </div>
    &lt;/html&gt;
</div>
<br></br>

:::tip[توجه]
یه چیزی که باید بدونی اینه که همه چیزی که داخل بخش `<body>` قرار می‌گیره، توی مرورگر نشون داده می‌شه. مثلاً عکس‌ها، متن‌ها، جداول، و هر چیز دیگه‌ای که توی صفحه می‌بینیم. اما عنصر `<title>` فقط عنوان صفحه رو نشون می‌ده، که معمولاً توی تب مرورگر یا بالای صفحه می‌باشد.
:::

## تاریخچه HTML
از اوایل دوران وب جهانی، چندین نسخه از HTML وجود داشته:

| سال | نسخه |
|-----|-------|
| 1989 | Tim Berners-Lee وب رو اختراع کرد |
| 1991 | Tim Berners-Lee HTML رو اختراع کرد |
| 1993 | Dave Raggett HTML+ رو معرفی کرد |
| 1995 | گروه کاری HTML HTML 2.0 رو تعریف کرد |
| 1997 | W3C توصیه کرد: HTML 3.2 |
| 1999 | W3C توصیه کرد: HTML 4.01 |
| 2000 | W3C توصیه کرد: XHTML 1.0 |
| 2008 | پیش‌نویس اول HTML5 از WHATWG منتشر شد |
| 2012 | HTML5 به عنوان استاندارد زنده توسط WHATWG تایید شد |
| 2014 | W3C توصیه کرد: HTML5 |
| 2016 | W3C توصیه نامزد کرد: HTML 5.1 |
| 2017 | W3C توصیه کرد: HTML5.1، ویرایش دوم |
| 2017 | W3C توصیه کرد: HTML5.2 |
